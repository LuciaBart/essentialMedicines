# A POLICY FOR DELIVERY OF ESSENTIAL MEDICINES TO VULNERABLE POPULATION IN ARGENTINA: A CASE STUDY OF THE REMEDIAR PROGRAM 

### Data and scripts underlying the findings described in: A Policy For Delivery Of Essential Medicines To Vulnerable Population In Argentina: A Case Study Of The REMEDIAR Program 

This repository contains the data and scripts required to reproduce the findings underlying the REMEDIAR program case study.

## Overview
The project is structured as a reproducible workflow based on secondary data. The pipeline analyses funding, procurement, distribution and Out-of-Pocket spending and financial protection. All analyses were implemented using the R Programming Language.


## Repository Structure
The repository is organized to ensure that every figure in the paper can be traced back to its specific source code:

[/Data:](https://github.com/LuciaBart/essentialMedicines/tree/main/Data) Contains secondary data used for analysis.

[/Scripts:](https://github.com/LuciaBart/essentialMedicines/tree/main/Scripts) Each script is named like the Figure it reproduces, for each section.


## Instructions for Reproduction

To replicate the study results, follow the steps below:

1. Clone or Download this repository to your local environment.

2. Ensure that R is installed.

3. Install the required library dependencies.

4. Run this command in R console:

`
source("RUN_PROCESS.r")
`
